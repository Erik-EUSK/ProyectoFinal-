Los documentos rtf,de texto enriquecido, tienen la posibilidad de dividir documentos en apartados o listas sin numerar.Aqui aparece algun ejemplo de listas:

Tipo de lista numerada.

  1. Elemento de nivel cero,cero
      2. 1.Elemento de nivel uno,cero
  2. 2.elemento de nivel uno,uno
      3. 1.elemento de nivel dos,cero
  3. 2.elemento de nivel dos,uno

tipo de lista no numerada

  • elemento de nivel cero,cero
    • elemento de nivel uno cero
      • elemento de nivel dos,cero
      • elemento de nivel dos,uno
    • elemento de nivel cero,uno
  • elemento de nivel cero,uno
    • elemento de nivel uno de cero
